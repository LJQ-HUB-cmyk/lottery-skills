# 双色球游戏规则

## 第一章 总则

### 第一条
根据《彩票管理条例》、《彩票管理条例实施细则》、《彩票发行销售管理办法》等有关规定，制定本规则。

### 第二条
中国福利彩票双色球游戏（以下简称双色球游戏）由中国福利彩票发行管理中心（以下简称中福彩中心）发行和组织销售，由各省、自治区、直辖市福利彩票销售机构（以下简称各省福彩中心）在所辖区域内销售。

### 第三条
双色球游戏采用计算机网络系统发行，在各省福彩中心设置的福利彩票销售网点销售，定期开奖。

### 第四条
双色球游戏实行自愿购买，凡购买者均被视为同意并遵守本规则。

### 第五条
不得向未成年人销售彩票或兑付奖金。

---

## 第二章 投注

### 第六条 基本投注规则
双色球游戏投注区分为红色球号码区和蓝色球号码区：
- **红色球号码区**：由1-33共三十三个号码组成
- **蓝色球号码区**：由1-16共十六个号码组成
- **单式投注**：选择6个红色球号码和1个蓝色球号码组成一注
- **每注金额**：人民币2元

### 第七条
购买者可在各省福彩中心设置的福利彩票销售网点投注，并获得经彩票投注专用设备打印出的双色球彩票。

### 第八条 投注方式
- **机选号码投注**：由彩票投注专用设备随机产生投注号码
- **自选号码投注**：由购买者指定投注号码

### 第九条 复式投注
复式投注是指所选号码个数超过单式投注的号码个数，具体规定如下：

#### （一）红色球号码复式投注
从红色球号码中选择7个以上（含7个）号码，从蓝色球号码中选择1个号码，组合为多注单式投注号码。

#### （二）蓝色球号码复式投注
从红色球号码中选择6个号码，从蓝色球号码中选择2个以上（含2个）号码，组合为多注单式投注号码。

#### （三）全复式投注
从红色球号码中选择7个以上（含7个）号码，从蓝色球号码中选择2个以上（含2个）号码，组合为多注单式投注号码。

### 第十条 胆拖投注
胆拖投注指先选取少于单式投注号码个数的号码作为每注都有的胆码，再选取除胆码以外的号码作为拖码，胆码与拖码个数之和必须多于单式投注号码个数。

#### （一）单式胆拖投注
- 从红色球号码中选择1至5个号码为胆码
- 再选取除胆码以外的号码作为拖码（胆码和拖码个数之和必须≥7个）
- 从蓝色球号码中选择1个号码

#### （二）复式胆拖投注
- 从红色球号码中选择1至5个号码为胆码
- 再选取除胆码以外的号码作为拖码（胆码和拖码个数之和必须≥7个）
- 从蓝色球号码中选择2个以上（含2个）号码

### 第十一条 倍投规则
- 投注倍数范围：2-99倍
- 单张彩票的投注金额最高不得超过20000元

### 第十二条 销售频率
双色球游戏按期销售，每周销售三期，期号按当年开奖次数排序。

### 第十三条
若因销售终端故障、通讯线路故障和不可抗力等原因造成投注不成功的，应退还购买者投注金额。

## 第三章 设奖

### 第十四条 奖金分配
双色球游戏按当期销售额分配如下：
- **51%** - 彩票奖金
  - 49% - 当期奖金
  - 2% - 调节基金
- **13%** - 彩票发行费
- **36%** - 彩票公益金

### 第十五条
双色球游戏采取全国统一奖池计奖。

### 第十六条 奖级设置
- **浮动奖级**：一等奖、二等奖
- **固定奖级**：三等奖、四等奖、五等奖、六等奖
- **特别规定期间**：固定奖级增设福运奖

### 第十七条 未执行特别规定期间的奖金规定

#### 一等奖
**奖池资金 < 1亿元时：**
- 奖金总额 = 当期浮动奖级奖金的75% + 奖池中累积的资金
- 单注最高限额：500万元

**1亿元 ≤ 奖池资金 < 15亿元时：**
- 第一部分：当期浮动奖级奖金的55% + 奖池累积资金，单注最高限额500万元
- 第二部分：当期浮动奖级奖金的20%，单注最高限额500万元

#### 二等奖
- 奖金总额 = 当期浮动奖级奖金的25%
- 单注最高限额：500万元

#### 固定奖级
| 奖级 | 单注奖金 |
|------|----------|
| 三等奖 | 3000元 |
| 四等奖 | 200元 |
| 五等奖 | 10元 |
| 六等奖 | 5元 |

### 第十八条 执行特别规定期间的奖金规定
**触发条件**：奖池资金 ≥ 15亿元时开始执行
**停止条件**：某期开奖后奖池资金 < 3亿元时停止执行

#### 一等奖
- 第一部分：奖池中累积的资金，单注最高限额500万元
- 第二部分：当期浮动奖级奖金的20%，单注最高限额500万元

#### 二等奖
- 奖金总额 = 当期浮动奖级奖金的80%
- 单注最高限额：500万元

#### 固定奖级
| 奖级 | 单注奖金 |
|------|----------|
| 三等奖 | 3000元 |
| 四等奖 | 200元 |
| 五等奖 | 10元 |
| 六等奖 | 5元 |
| **福运奖** | 5元 |

### 第十九条 封顶规定

#### 一等奖封顶
- 当期一等奖奖金总额超出1亿元时触发
- 触发后单注奖金 = 1亿元 / 当期一等奖实际中出注数
- 超出封顶部分资金转入奖池

#### 二等奖封顶
- 当期二等奖奖金总额超出7000万元时触发
- 触发后单注奖金 = 7000万元 / 当期二等奖实际中出注数
- 超出封顶部分资金转入奖池

### 第二十条 保底规定

#### 一等奖保底
- 若一等奖单注奖金 < 二等奖单注奖金的2倍 且 < 500万元
- 由调节基金补足至二等奖单注奖金的2倍（不超过500万元）
- 若二等奖单注奖金 < 6000元时，一等奖保底为12000元

#### 二等奖保底
- 若二等奖单注奖金 < 三等奖单注奖金（3000元）的2倍
- 由调节基金补足为6000元

**注意**：若保底与封顶同时触发，优先确保奖金总额封顶。

### 第二十一条 奖池资金组成
奖池资金由以下部分组成：
- 未中出的浮动奖级奖金
- 超出单注奖金封顶限额部分的奖金
- 超出一等奖、二等奖总奖金封顶的奖金

### 第二十二条 调节基金
调节基金包括：
- 按销售总额的2%提取部分
- 奖金按元取整后的余额

**用途**：
- 支付不可预见的奖金风险支出
- 设立特别奖

### 第二十三条
双色球游戏的当期奖金和奖池资金不足以兑付当期中奖奖金时，由调节基金补足。

### 第二十四条 风险控制
双色球游戏每期开奖时对当期最大返奖总额实行限额赔付风险控制，若当期应付奖金超过当期最大返奖总额，则相应调整相关奖级单注奖金。

## 第四章 开奖

### 第二十五条 开奖时间
双色球游戏由中福彩中心统一开奖，**每周二、四、日开奖**（休市日除外）。

### 第二十六条 开奖流程
1. 公证人员封存销售数据资料
2. 在公证人员监督下通过摇奖器确定开奖号码
3. 先摇出6个红色球号码
4. 再摇出1个蓝色球号码

### 第二十七条 信息公布
每期开奖后，中福彩中心和各省福彩中心应向社会公布：
- 开奖号码
- 当期销售总额
- 各奖级中奖情况
- 奖池资金余额等信息
- 并将开奖结果通知销售网点

---

## 第五章 中奖

### 第二十八条 中奖规则
根据购买者所选单式投注号码与当期开奖号码的相符情况（顺序不限），确定相应的中奖资格：

| 奖级 | 中奖条件 |
|------|----------|
| **一等奖** | 6个红球 + 1个蓝球 全部相同 |
| **二等奖** | 6个红球 相同 |
| **三等奖** | 5个红球 + 1个蓝球 相同 |
| **四等奖** | 5个红球 相同 或 4个红球 + 1个蓝球 相同 |
| **五等奖** | 4个红球 相同 或 3个红球 + 1个蓝球 相同 |
| **六等奖** | 1个蓝球 相同 |
| **福运奖** | 3个红球 相同（特别规定期间） |

### 第二十九条 中奖原则
当期每注投注号码只兑付所中得最高奖级奖金，不能兼中兼得（派奖活动中特别奖除外）。

---

## 第六章 兑奖

### 第三十条 兑奖凭证
- 中奖彩票为兑奖唯一凭证
- 中奖彩票因玷污、损坏等原因不能正确识别的，不能兑奖

### 第三十一条 兑奖期限
- 双色球游戏兑奖当期有效
- 中奖者应当自开奖之日起**60个自然日内**兑奖
- 逾期未兑奖视为弃奖，弃奖奖金纳入彩票公益金

### 第三十二条 身份验证
兑奖机构可以查验中奖者的中奖彩票及有效身份证件，中奖者兑奖时应予配合。

---

## 第七章 附则

### 第三十三条
本规则自批准之日起4个月内执行，具体执行时间由中福彩中心及时向社会公告。
