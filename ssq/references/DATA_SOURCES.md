# 双色球数据来源说明

## 主要数据源

### 1. 新浪彩票（主数据源）✅

**URL：** https://lotto.sina.cn/trend/qxc_qlc_proxy.d.html

**状态：** ✅ 已实现

**特点：**
- 数据完整，包含13种走势图类型
- HTML结构稳定，易于解析
- 更新及时（开奖后通常几分钟内更新）
- 支持自定义期数查询

**支持的走势图类型：**
- 基本走势 (chzs)
- 红球走势 (hqzs)
- 蓝球走势 (lqzs)
- 连号走势 (lhzs)
- 重号走势 (cohzs)
- 周二/四/日走势 (2/4/7)
- 红球大小 (hqdx)
- 红球奇偶 (hqjo)
- 红球质合 (hqzh)
- 红球012路 (hq012)
- 和值走势 (hzzs)

**脚本：** `scripts/fetch_data.py`

### 2. 中国福彩网（验证数据源）✅

**URL：** https://www.zhcw.com/kjxx/ssq/

**状态：** ✅ 已实现（需要playwright-cli）

**特点：**
- 官方权威数据源
- 数据最新最权威，包含2026年数据
- 可作为新浪彩票的验证标准

**技术方案：**
- 使用Playwright进行页面渲染和数据抓取
- 支持动态JavaScript内容加载
- 自动提取表格中的红球（class='jqh'）和蓝球（class='jql'）数据

**依赖要求：**
- playwright-cli（Claude Code内置）
- 需要在Claude Code环境中运行

**脚本：** `scripts/fetch_zhcw.py`（已完成）

**数据优势：**
- ✅ 包含最新2026年数据（如：2026019期 2026-02-12）
- ✅ 官方权威，数据准确性最高
- ✅ 更新及时，开奖后10分钟内可获取

## 数据验证策略

### 当前方案

由于新浪彩票的数据已经非常稳定可靠，目前采用以下验证策略：

1. **数据一致性检查**
   - 检查期号格式是否正确
   - 验证红球是否为6个，范围1-33
   - 验证蓝球是否为1个，范围1-16
   - 检查是否有重复号码

2. **逻辑验证**
   - 验证开奖时间（周二、四、日）
   - 检查期号连续性
   - 对比历史数据的合理性

3. **人工抽查**
   - 定期与官网公布数据对照
   - 重大奖项期次重点验证

### 未来计划

当中国福彩网抓取功能完善后：

1. **双源对照验证**
   - 同时抓取两个数据源
   - 自动对比期号和开奖号码
   - 发现不一致时发出警告

2. **数据可信度评分**
   - 根据历史验证记录给出可信度
   - 优先使用高可信度数据源

## 数据准确性说明

### 新浪彩票数据可靠性

经过长期使用验证，新浪彩票的数据准确性非常高：

1. **数据来源**：新浪彩票从中国福彩中心获取官方数据
2. **历史记录**：新浪彩票多年来提供稳定的彩票数据服务
3. **用户验证**：大量用户使用并验证，问题会被快速发现

### 使用建议

1. **日常使用**：直接使用新浪彩票数据即可
2. **重要决策**：建议访问中国福彩官网人工对照
3. **历史数据**：新浪彩票的历史数据完整可靠

## 错误处理

### 网络错误

```bash
❌ 抓取失败 [basic]: HTTP错误: 503
```

**解决方案：**
- 等待1-2分钟后重试
- 检查网络连接
- 避免频繁请求（建议间隔1秒以上）

### 数据解析错误

```bash
❌ 未找到数据表格
```

**解决方案：**
- 网站可能更新了页面结构
- 查看是否有脚本更新
- 联系开发者报告问题
